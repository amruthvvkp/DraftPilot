# DraftPilot data layer (SQLModel) — sub-agent

Full checklist: **`.claude/skills/sqlmodel-data/SKILL.md`** (skill id: `sqlmodel-data`).
Scaffolding one model end-to-end: skill `add-model`.

Key contract: Postgres + asyncpg (async app engine) with Alembic. Use SQLModel's
`AsyncSession` (`.exec()`), `col()` for typed-column queries, timezone-aware timestamps via
`sa_type=DateTime(timezone=True)`. Alembic `env.py` uses `settings.postgres.sync_dsn` and
targets `SQLModel.metadata` (import `draftpilot.models` to register tables). Always review
autogenerated revisions before committing.
